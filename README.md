# JP Bestseller Dashboard

일본 제품 YouTube 트렌드 분석, Gemini 딕셔너리 랭킹, 네이버 쇼핑 최저가 비교 Streamlit 대시보드입니다.

## 로컬 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud Secrets

```toml
YOUTUBE_API_KEY = "..."
GEMINI_API_KEY = "..."
NAVER_CLIENT_ID = "..."
NAVER_CLIENT_SECRET = "..."
```
